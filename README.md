speech2logs
==============================

Convert your voice to logs in text after every day with speech2logs!

Project Organization
------------

```
speech2logs/
├── LICENSE     
├── README.md                  
├── Makefile                     # Makefile with commands like `make data` or `make train`                   
├── configs                      # Config files (models, training hyperparameters, API keys, bot settings)
│   ├── model1.yaml              
│   ├── whisper.yaml             # Whisper config file
│   ├── llama3.yaml              # Llama3 config file
│   └── telegram_bot.yaml        # Telegram bot configuration
│
├── data                         
│   ├── external                 # Data from third party sources.
│   ├── interim                  # Intermediate data that has been transformed.
│   ├── processed                # The final, canonical data sets for modeling.
│   ├── raw                      # The original, immutable data dump.
│   └── json_logs                # JSON logs from the Telegram bot.
│       ├── 2024-11-18.json      # Example JSON file for logs.
│
├── docs                         # Project documentation.
│
├── models                       # Trained and serialized models.
│   ├── whisper                  # Whisper models and checkpoints.
│   └── llama3                   # Llama3 models and checkpoints.
│
├── notebooks                    # Jupyter notebooks.
│
├── references                   # Data dictionaries, manuals, and all other explanatory materials.
│
├── reports                      # Generated analysis as HTML, PDF, LaTeX, etc.
│   └── figures                  # Generated graphics and figures to be used in reporting.
│
├── requirements.txt             # The requirements file for reproducing the analysis environment.
└── src                          # Source code for use in this project.
    ├── __init__.py              # Makes src a Python module.
    │
    ├── data                     # Data engineering scripts.
    │   ├── build_features.py    
    │   ├── cleaning.py          
    │   ├── ingestion.py         
    │   ├── labeling.py          
    │   ├── splitting.py         
    │   └── validation.py        
    │
    ├── models                   # ML model engineering (a folder for each model).
    │   ├── model1      
    │   │   ├── dataloader.py    
    │   │   ├── hyperparameters_tuning.py 
    │   │   ├── model.py         
    │   │   ├── predict.py       
    │   │   ├── preprocessing.py 
    │   │   └── train.py         
    │   │
    │   ├── whisper              # Scripts for Whisper inference
    │   │   ├── inference.py     # Whisper inference logic
    │   │   └── preprocessing.py # Preprocessing for Whisper (e.g., audio transformations)
    │   │
    │   └── llama3               # Scripts for Llama3 inference
    │       ├── inference.py     # Llama3 inference logic
    │       └── preprocessing.py # Preprocessing for text generated by Whisper
    │
    ├── bot                      # Telegram bot integration
    │   ├── __init__.py          # Makes bot a Python module.
    │   ├── bot_handler.py       # Main script for interacting with the Telegram API
    │   ├── message_parser.py    # Parse messages from Telegram
    │   └── json_saver.py        # Save the parsed messages into JSON format
    │
    ├── visualization            # Scripts to create exploratory and results-oriented visualizations.
    │   ├── evaluation.py        
    │   └── exploration.py  
    │
    └── utils                    # Utility functions shared across the project.
        ├── file_utils.py        # File handling utilities.
        ├── json_utils.py        # Utilities for handling JSON logs.
        ├── date_utils.py        # Utilities for handling date and time formats.
        └── telegram_utils.py    # Helper functions for Telegram API interactions. 
```
